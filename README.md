# R9 Motion — native animated wallpapers for macOS

Living, GPU-rendered wallpapers — fluid ribbons, flowing waves, rain ripples
and breathing echo rings — as your macOS desktop background. Born from the
R9-EQ / R9-Music fluid artwork engine.

> Created by **Rhineul Islam** · free for the community · v0.5.0

**Website:** https://xpro-gamer-rhine.github.io/R9-Motion-app/

## Install (one line)

```bash
curl -fsSL https://raw.githubusercontent.com/XPro-Gamer-Rhine/R9-Motion-app/main/install.sh | bash
```

A wave icon appears in the menu bar — everything is configured from there.

## Features

- **Four animations**: Fluid (the R9 signature look), Waves, Ripple, Reverb.
- **Make it yours**: three custom colors or seeded palettes, speed, scale,
  and a Shuffle Pattern dice for endless variations.
- **Multi-monitor**: one synced look everywhere, or an individual wallpaper
  per display, with copy-to-all.
- **Featherweight**: ~36 MB memory, ~2–3% CPU at 30 fps — renders at reduced
  resolution the GPU upscales (invisible on these soft patterns), and pauses
  completely when a fullscreen app covers the desktop. 24/30/60 fps + Eco/
  Balanced/Sharp quality controls.
- **Native only**: AppKit + Metal + SwiftUI. No Electron, no web views,
  no dependencies. The whole app is ~120 KB.
- **Start at login** toggle built in.

## Requirements
macOS 13+ · Apple Silicon or Intel.
